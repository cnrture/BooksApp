BooksApp with MVVM & Retrofit & Navigation
</h1>

<h2 align="center">
Project Features
</h2>

<h3 align="left">
 - MVVM (Model, View, ViewModel) <br>
 - LiveData <br>
 - DataBinding <br>
 - Retrofit <br>
 - BottomNavigation <br>
 - ViewPager2 in TabLayout  <br>
 - SearchView in Toolbar <br>
 - Picasso for images 
 </h3><br>

<h2 align="center">
Main Screen
</h2>

<h3 align="left">
- Books listed using RecyclerView + DataBinding. <br>
- Book search feature works with author, book name and publisher name. Search can be made with the SearchView in the Toolbar or the ChipButtons at the top of the page. <br>
- When the book image is pressed, the CustomAlertView opens and details are displayed. The book is sent to the basket with the Add to Cart button. <br>
- The "+" button in the Toolbar is used to add a book. (Note: Images are added with url.) <br>
 </h3>
